This directory contains the up-to-date version of then-player computer game class hierarchy. The base class isComputerGame. Programs that use this library are soft-linked to its nplayer.jar.Whenever a change is made to one of the source files in this directory,all of the Java source should be recompiled and a new jar file shouldbe made:  rm *.class  // Remove all class files  rm nplayer.jar // Remove the old jar file  javac *.java // Compile all Java source files  jar cvf nplayer.jar *.class  // Create the new jarTests of the library should be performed from within itssibling directories. Here's a list of the directories thatpresently use this library:  showdown  slidingtileCLUI  slidingtileGUI  twentyone  wordguess
